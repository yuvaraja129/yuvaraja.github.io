SIC Python Script

This project involved developing a Python script to automate value modification in an Excel sheet, specifically organizing company SIC codes for a marketing research project my mentor was working on. I utilized Python libraries, including pandas, to achieve these results.

  • I improved my technical knowledge of using Python to modify Excel sheets.
  • I became more familiar with Pandas for efficient data handling.
  • I practiced my algorithmic skills, reducing script runtime from around 10 minutes to approximately 30 seconds through more efficient algorithms.
